# Contributing

Danke, dass du Eventlotse verbessern möchtest.

## Entwicklung

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Coding Standards

- TypeScript bevorzugen.
- Komponenten klein und fachlich benennen.
- UI immer responsiv denken.
- Keine sensiblen Daten in Commits.
- Neue Features möglichst mit klarer Datenstruktur und späterer Backend-Fähigkeit bauen.

## Pull Requests

- Beschreibe das Problem und die Lösung knapp.
- Führe aus, welche Tests oder Checks gelaufen sind.
- Aendere Lizenz- und Rechtsdokumente nur bewusst.
