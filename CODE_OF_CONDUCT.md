# Code of Conduct

Dieses Projekt soll freundlich, sachlich und hilfreich bleiben.

Erwartet wird:

- Respektvoller Umgang.
- Konstruktive Kritik.
- Keine Belästigung, Diskriminierung oder persönlichen Angriffe.
- Fokus auf Lösungen und nachvollziehbare Entscheidungen.

Projektbetreiber können Beiträge entfernen, die diesen Grundsätzen widersprechen.
